# Sky Hopper 🚀☁️

**Sky Hopper** is a fast-paced, challenging 2D platformer game built with Python and Pygame. Embark on an epic journey across 10 unique, gravity-defying levels and test your reflexes to reach the exit door! 🏃‍♂️💨

## 🎮 Game Features
*   **10 Challenging Levels:** Progress through increasingly difficult stages with complex platform layouts and shifting obstacles. 🏗️
*   **Dynamic Movement:** Master the timing with moving platforms and utilize the **Double Jump** mechanic to navigate tricky gaps! ⚡️
*   **Collect & Unlock:** Gather all the golden coins scattered across the map to activate the exit portal. 🪙✨
*   **Speedrun Mode:** Race against the clock and try to beat your personal best time in every session! ⏱️🏆
*   **Immersive Experience:** Enjoy smooth character animations, immersive sound effects, and a lively atmosphere. 🎮🎵

## 🕹️ How to Play
*   **Move:** Use the **Left/Right Arrow Keys** to run.
*   **Jump:** Press **Space** to jump. (You can perform a **Double Jump** in mid-air!)
*   **Objective:** Collect all the coins in the level, then reach the portal to advance.
*   **Warning:** Watch your step! If you fall into the void, it's Game Over. ⚠️

## 🛠️ Built With
*   **Language:** Python
*   **Engine:** Pygame
*   **Focus:** Collision physics, sprite animation, and state-based game design.

## 🚀 How to Run
1. Ensure you have Python installed.
2. Install Pygame:
   ```bash
   pip install pygame
   ```
3. Run the game:
   ```bash
   python main.py
   ```
